An interactive product experience for the Epomaker Galaxy65 mechanical keyboard.  
Scroll-driven 4K frame animation, switch acoustics engine, real-time configurator, and zero-bloat architecture.

**[Live Demo →] (https://epomaker.vercel.app)**

---

### Spec Sheet

| Feature | Implementation |
|---|---|
| **Scroll Engine** | 219-frame 4K canvas sequence with LERP interpolation |
| **Switch Acoustics** | Web Audio API synthesizer + physical press animations |
| **Keycap Configurator** | Live DOM-swapping visual matrix |
| **Navigation** | IntersectionObserver scrollspy + cinematic warp scrolling |
| **Mobile** | Terminal-style drawer, responsive bento grid |

---

### Run Local

```bash
git clone https://github.com/ShayanGaba/Epomaker-Keyboard-Website.git
cd Epomaker-Keyboard-Website
npm install
npm run dev
```
